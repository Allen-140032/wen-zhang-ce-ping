# Contributing Guide

感谢你对 AI爆款潜力测评器 的关注！

## 如何贡献

### 报告 Bug

1. 在 [Issues](https://github.com/Allen-140032/wen-zhang-ce-ping/issues) 搜索是否已有相同问题
2. 如果没有，新建 Issue，选择 Bug 模板
3. 描述：做了什么、期望结果、实际结果、浏览器版本

### 提建议

1. 新建 Issue，选择 Feature Request 模板
2. 说清楚：你想加什么功能、为什么有用、怎么用

### 提交代码

1. Fork 本仓库
2. 创建分支：`git checkout -b feature/你的功能名`
3. 修改代码
4. 确保单文件 `index.html` 仍可正常运行（双击打开测试）
5. 提交 PR，说明改动内容和测试方式

### 代码规范

- 保持单文件结构（所有代码在 `index.html` 内）
- 手机优先，确保在微信浏览器内正常显示
- 不引入外部依赖（CDN 除外）
- 中文注释

## 行为准则

- 尊重所有参与者
- 建设性反馈 > 单纯批评
- 目标是做出好用的工具

## 开发环境

只需要一个浏览器和文本编辑器。双击 `index.html` 即可启动。
