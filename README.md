# 公众号爆款潜力测评器 MVP

## 交付物

- `index.html`：纯前端单页工具，手机端适配。

## 功能

- 粘贴公众号标题、开头或正文。
- 输出 6 个维度评分：标题吸引力、开篇钩子、段落节奏、情绪曲线、可读性、行动号召。
- 输出总分、等级判断、综合建议。
- 支持本地演示评分，便于无 API Key 时验证流程。
- 支持 DeepSeek、API2D / GPT-4o、自定义 OpenAI-Compatible 接口。

## 使用方式

1. 直接双击打开 `index.html`。
2. 默认使用“本地演示评分”，无需 API Key。
3. 如需 API 评分，选择 DeepSeek 或 API2D，输入 API Key 后点击“开始测评”。

## 注意事项

- 这是纯前端 MVP。API Key 会在浏览器里用于本次请求，不会写入本地存储。
- 如果公开部署，不建议让用户在前端直接填写真实 Key；后续应改成 Friday 中转接口。
- 不依赖服务器、不依赖数据库、不做用户登录。
